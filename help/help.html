These are scripts for Substance Painter 2018 done with Autohotkey, that automaticaly moves your mouse. They are searching for some image patterns on your the screen and are trying to find the options to automatically click on.

 

 

Features so far:

  • .toggle pressure for size and flow

  • .set Alignment for brush to UV or Tnagent|Wrap

  • .custom predefined searches into the Shelf with these you can quickliy acces some favorites tools or brushes

  • .quckly create some favorites effects like: Blur or Sharpen....

  • .all these options can be accesd by marking Menus or windows with shortcuts

  • .incrementing slider

 

Available shortcuts:

  • Ctrl + Alt + A :toogle pressure for Flow

  • Ctrl + Alt + S :toogle pressure for Size

  • Alt + Left Arrow :incremnet slider's value by a custom set value

  • Alt + Right Arrow :decrease slider's value by a custom set value

  • Alt + Shift + Left Arrow :set custom value for the increment

  • Middle Mouse Pressed:PIE MENUS:

    Each Pie menu can be accesed by pressing it's shortcut and the Middle Mouse

    Don't worry if the shortcuts for the Pie Menius are the same as the ones in S.Painter, they will still work in S.Painter but on the release of the key. You can customize each Pie menu by adding buttons like in the Shortcuts window
